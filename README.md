# NCAA Baseball Camps 2026 Directory

A high-fidelity project to compile and present camp information for over 550 NCAA Division I and Division II baseball programs.

## Features

- **V6 Ultra-Fidelity Extraction**: Intelligent, multi-engine scraping with built-in date range detection (Jan-Dec 2026).
- **Mascot-Based Search**: High-accuracy targeting using team mascots and university aliases.
- **Premium Directory**: Reactive, searchable HTML interface with expandable camp details.
- **Micro-Animation & UX**: Custom Vanilla CSS directory with smooth transitions and interactive cards.
- **Watchdog Stability**: Automated process monitor with self-healing restart capabilities.

## Tech Stack

- **Node.js**: Scraping and data processing (Puppeteer).
- **Vanilla JS/HTML/CSS**: Frontend directory.
- **Git**: Version control (Conventional commit standard).

## Project Structure

- `extract_camp_details.js`: Main V6 extraction engine.
- `camps_data.json`: Master project database.
- `tasks.md`: Current project backlog and status tracking.
- `index.html`: The live production directory.
